NOTE:
-----

All the native code of the codec is extracted from the open source code of the Android platform.
Then there is a JNI interface which permits to access to these codecs from Java side.



COMPILATION PROCEDURE:
----------------------

copy Android.mk, avc_h263 and m4v_h263 to <ndk-root>/apps/NativeLib/project/
copy Application.mk to <ndk-root>/apps/NativeLib/

Build codecs:

make APP=NativeLib

Out directory:

<ndk-root>/apps/NativeLib/project/libs/armeabi
