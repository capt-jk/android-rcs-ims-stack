README - RCS API Demos


Demos:
------

/extension	Demonstrates how to support new RCS-e extensions or capabilities dynamically on top of the RCS-e stack.

/gsma		Demonstrates how to implement GSMA UI connectors specification which permits to link RCS-e application with other applications.

/sip		Demonstrates how to deploy new SIP services (session oriented services or instant message oriented services) on top of the RCS-e stack.
